🔄 DATA FLOW
═══════════════════════════════════════════════════════════════════

Parent Component
    ↓ (cards prop)
MoveableDashboard
    ↓ (card prop via scoped slot)
DashboardCard
    ↓ (default slot)
User Content Component
    
User Interaction → Event Emission → Parent Handler → State Update
